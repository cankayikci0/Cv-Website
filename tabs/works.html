HIT THE SPACE INVADERS

This game is written with OpenGL and GLUT libraries of C. The game starts with 'F1'. If you press F1 while game is running' the game stops. Pressing 'F3' shows the scoreboard. Your scores are being showed in an ascending order in the scoreboard. Spacebar is used to fire. Right and Left keys are used to move horizontally. The main of the game is to shoot the enemies. To earn points, you need to shoot down enemies. Click here to reach out the source code.

SUPERMEN

This game is written with OpenGL and GLUT libraries of C. The main purpose of the game is to place the supermen where you want and watch them fly. To place the supermen, you need to click somewhere on the screen. You can create maximum 10 supermen. Each supermen will have different cape colors. To fly supermen, you can press 'F1'. The screen changes when the program is running.
